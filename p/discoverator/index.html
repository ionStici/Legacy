Welcome! Challenge your geography knowledge with Discoverator! An interactive app with real world information.

By clicking the button on the right: The data about the country is displayed using the geolocation API + geocoding API + restcountries API. Note that you have to allow the browser to read your location data in order to render the box.

The initial purpose of the respective application was: practicing web development skills. The app is build entirely in HTML, CSS (+Sass) and JavaScript, no backend or frontend libraries involved.

Concepts used for building the application: Frontend knowledge, Leaflet Library, APIs, Sass, localStorage, and ...
